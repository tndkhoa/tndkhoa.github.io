Excel VLOOKUP Function

The VLOOKUP function in Excel is used to search for a value in the first column of a table and return a corresponding value from another column.

Syntax

VLOOKUP(lookup_value, table_array, col_index_num, [range_lookup])

Example

Consider the following table:

Product ID Product Name Price
101 Apple $1.00
102 Banana $0.50
103 Orange $0.75

To find the price of a product with ID 102, use the formula:

=VLOOKUP(102, A2:C4, 3, FALSE)

This will return $0.50 since Banana has ID 102.
